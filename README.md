## 安装教程

### 方式：直接安装

1. 下载此扩展到任意位置
2. 安装SimMusic后，打开「扩展」页面，将此扩展拖入扩展页面后点击「确认」按钮安装

## 扩展说明

1. 若扩展存在Bug请及时[提交](https://github.com/PYLXU/SMEx_playerPage/issues)，非常感谢
2. 必须同时启用官方设置项：播放页深色模式
